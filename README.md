# Dự án Quản lý Nhân sự

Dự án ứng dụng web hỗ trợ quản lý thông tin nhân sự, đối tác và khách hàng. Hệ thống sử dụng trực tiếp các file Excel (.xlsx) để lưu trữ dữ liệu thay vì SQL, giúp dễ dàng cài đặt và vận hành.

## Chức năng chính
- Quản lý danh sách nhân viên, khách hàng và đối tác.
- Hỗ trợ các thao tác cơ bản: xem, thêm, sửa, xóa dữ liệu.
- Đồng bộ và lưu trữ dữ liệu trực tiếp vào file Excel.

## Công nghệ sử dụng
- Ngôn ngữ: Python
- Web Framework: Flask
- Front-end: HTML, CSS
- Thư viện xử lý dữ liệu: Chi tiết trong file requirements.txt
